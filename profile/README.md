# 🐾 Team6 — Cloud Architecture Payment System

> 강아지 용품 이커머스 **견생보감**을 개발한 6인 팀입니다.  
> 단 한 번의 끊김 없는 결제 경험을 목표로, 클라우드 기반 결제 시스템을 설계하고 구현했습니다.

<br>

## 📦 프로젝트

| 프로젝트 | 설명 | 링크 |
|----------|------|------|
| 🛒 **견생보감** | 강아지 용품 이커머스 — PortOne 결제 연동, 주문/환불 관리, JWT 인증 | [payment-system](https://github.com/Team6-Cloud-Architecture-Payment-System/payment-system) |
| 🌐 **배포 URL** | 운영 서버 | https://api.dogpedia.store |

<br>

## 🛠 기술 스택

| 영역 | 기술 |
|------|------|
| **Frontend** | Vanilla JavaScript, HTML, CSS, PortOne SDK |
| **Backend** | Java 17, Spring Boot, Webhook |
| **DB / 인프라** | MySQL (AWS RDS), Redis, AWS EC2, AWS VPC |
| **네트워크** | Caddy (Reverse Proxy, HTTPS), 가비아 도메인 |
| **CI** | GitHub Actions |
